# 📊 Telco Customer Churn Analysis Dashboard

## Project Overview

This Power BI dashboard analyzes customer churn using the IBM Telco Customer Churn Dataset. It provides interactive insights into customer behavior, churn trends, revenue loss, and factors affecting customer retention.

## Dashboard Preview

![Dashboard](Pictures/dashboard.png)

## Key Performance Indicators (KPIs)

- Total Customers
- Total Churned Customers
- Churn Rate
- Average Monthly Charges
- Monthly Revenue Lost

## Dashboard Features

- Customer Churn by Contract Type
- Customer Churn by Tenure Group
- Customer Churn by Monthly Charges
- Customer Churn by Internet Service
- Customer Churn by Tech Support
- Revenue Lost by Contract Type
- Interactive Filters
- Reset Filters Button

## Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel

## Dataset

IBM Telco Customer Churn Dataset
